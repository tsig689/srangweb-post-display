Display WordPress posts with category filtering, pagination, view counts, and title-only mode.

== Description ==

Srangweb Post Display lets you show posts in cards or title-only lists with optional category filters.

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/srangweb-post-display`
2. Activate the plugin through the WordPress plugins screen
